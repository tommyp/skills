## Instructions

Ask for edit permissions if you don't have them already.

Using `gh`, look at the unresolved comments on this pull request, and summarize the feedback for the user. Decide if each comment warrants a change, and if so, what the change is.

Fix each issue if required and summarize the change you made. If you think a comment doesn't require a change, explain why. If there are no unresolved comments, say so.
